Dad Joke Generator - NET2008 Assignment 4


1.Description
--------------------

This is a simple Python Flask web application called "Dad Joke Generator".
It has two endpoints:

- "/"     : shows a basic HTML page with instructions.
- "/joke" : returns a random dad joke in JSON format.

The application calls the public icanhazdadjoke.com API to fetch
a random dad joke.

How to run the app locally


1. Make sure you have Python 3 installed.
2. Install dependencies:

   pip install -r requirements.txt

https://github.com/SahilvirDhillon/Simple-Dad-joke-Generator.git
